# Simulação de rompimento de barragem


## Introdução

As barragens são estruturas de concreto utilizadas de diversas formas em processos produtivos, energéticos, pesca e até mesmo recreação e lazer. Comumente no Brasil, devido à elevada disponibilidade de minérios e bacias hidrográficas, as barragens controlam o fluxo de água ou dejetos de forma artificial ou natural.
